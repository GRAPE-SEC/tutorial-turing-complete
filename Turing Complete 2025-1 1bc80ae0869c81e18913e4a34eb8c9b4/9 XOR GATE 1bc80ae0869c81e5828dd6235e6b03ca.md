# 9. XOR GATE

XOR 게이트는 게이트 중에서 제일 특이한 게이트인데, 이 게이트는 

이 게이트는 입력 두개가 서로 다른지 검사한다. **다르면 1 을 출력하고, 같으면 0 을 출력한다.**

진리표는 이렇다.

| input1 | input2 | output |
| --- | --- | --- |
| 0 | 0 | 0 |
| 1 | 0 | 1 |
| 0 | 1 | 1 |
| 0 | 0 | 0 |
