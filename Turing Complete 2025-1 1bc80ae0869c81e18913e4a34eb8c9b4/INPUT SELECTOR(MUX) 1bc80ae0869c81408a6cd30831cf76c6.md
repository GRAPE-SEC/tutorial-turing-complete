# INPUT SELECTOR(MUX)

입출력을 관찰해보면, 

- 두 입력 중 하나를 선택하여 출력으로 내보내고,
- 나머지 한 입력은 무시

이다.

따라서 논리 게이트는 필요하지 않고, “어떤 조건” 에 의해서 출력하거나 출력하지 않거나를 제어해야하므로 스위치가 필요함을 알 수 있다.

스위치의 컬렉터와 이미터를 연결하고 베이스에 제어 전압을 연결하면 된다.
