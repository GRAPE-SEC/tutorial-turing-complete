# Double The Number

# 묻고 따블로 가!

![image.png](/images/7_Double_The_Number/image.png)

이진수를 두배로 바꾸는 회로를 만들면 된다.

사실 생각해보면 쉬운데, 

- 14 의 두배는 28 이다.
- 14 = 8 + 4 = 1110
- 28 = 16 + 8 + 4 = 11100

즉, **그냥 한칸 왼쪽으로 밀면 된다.**
