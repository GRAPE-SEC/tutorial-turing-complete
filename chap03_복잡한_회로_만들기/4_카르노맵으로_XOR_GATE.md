# 수학

지금쯤이면 다들 눈치 챘을 건데, 사실 논리 게이트는 **수학 계산과 똑같다.**

- and : + (더하기)
- or : x (곱하기)

즉, 논리 회로는 **“1 과 0 만 가지고 더하기 곱하기를 하는 것”** 이다

XOR 게이트는 

```
(A x [B]) + ([A] x B)
```

를 계산한다.
